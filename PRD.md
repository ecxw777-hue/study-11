## 1. 문제 정의

### 1.1 현재 상황

한국에서 디지털 노마드로 생활하려는 사람들이 증가하고 있지만, **한국 도시별 노마드 생활 조건을 체계적으로 비교할 수 있는 전문 플랫폼이 존재하지 않는다.**

현재 노마드들은 다음과 같은 문제를 겪고 있다:

- **정보 파편화**: 생활비, 인터넷 속도, 안전, 날씨, 교통 등의 정보가 각기 다른 사이트에 분산되어 있어 도시별 비교가 어렵다.
- **신뢰도 부족**: 블로그나 SNS의 개인 후기는 주관적이고, 데이터 기반의 객관적 비교가 불가능하다.
- **접근성 부재**: Nomadlist.com 같은 글로벌 플랫폼은 한국 도시 데이터가 부족하고, 한국어를 지원하지 않으며, 한국 고유의 지표(KTX 접근성, 미세먼지, 한달살기 프로그램 등)를 다루지 않는다.

### 1.2 기회

- 한국 워케이션/한달살기 시장이 빠르게 성장 중 (지자체 지원 프로그램 증가)
- 디지털 노마드 관련 콘텐츠의 검색량 지속 증가
- 데이터 기반 도시 비교 플랫폼에 대한 수요 존재

## 3. 제안 해결책

### 3.1 제품 개요

**노마드코리아(Nomad Korea)** — 한국 디지털 노마드를 위한 도시 탐색 및 비교 플랫폼

홈페이지 한 화면에서 한국 50개 이상 도시의 노마드 생활 조건을 **데이터 기반으로 비교**할 수 있고, 커뮤니티 활동(밋업, 한달살기)까지 연결하는 올인원 허브 역할을 한다.

### 3.2 핵심 가치 제안

1. **데이터 기반 의사결정**: 150개 이상 지표로 도시를 객관적으로 비교
2. **한국 특화 지표**: KTX 접근성, PM2.5, 한달살기 프로그램 등 글로벌 플랫폼에 없는 정보
3. **직관적 UI**: 도시 카드 한 장에 핵심 정보를 모두 담아 빠른 비교 가능
4. **커뮤니티 연결**: 밋업, 한달살기 모집, 노마드 프로필 등 소셜 기능

### 4.4 섹션 04: 필터 & 정렬 바

```

║                                                                                              ║
║  SORT:  노마드스코어 ▾    │    VIEW:  ⊞ 그리드   🗺️ 지도   ☰ 리스트   📊 차트                ║
║                                                                                              ║
║  활성 필터:  [ 예산 ≤ 150만원 ✕ ]  [ 인터넷 ≥ 50Mbps ✕ ]      결과: 32개 도시  [ 초기화 ]    ║
║                                                                                              ║
╚══════════════════════════════════════════════════════════════════════════════════════════════════╝
  ▲ position: sticky · top: 56px (below nav) · z-index: 998 · 필터 변경 시 URL params 동기화
```

**기능 요구사항**:

| ID | 요구사항 | 우선순위 |
|----|---------|---------|
| FILTER-01 | 필터 항목: 예산, 인터넷, 날씨, 안전, 교통 (드롭다운) | P0 |
| FILTER-02 | "더보기" 클릭 시 추가 필터 노출 (미세먼지, 코워킹 수, 리뷰 수 등) | P2 |
| FILTER-03 | 정렬 옵션: 노마드스코어, 생활비(낮은순/높은순), 인터넷 속도, 안전, 평점 | P0 |

| 안전 | 10단계 바 그래프 | 경찰청 범죄율 통계 |
| 평점 | 사용자 리뷰 평균 + 리뷰 수 | 사용자 리뷰 |
| 날씨 | 현재 기온 / 체감온도 | 실시간 날씨 API |
| PM2.5 | 현재 초미세먼지 수치 | 에어코리아 API |
| KTX | 서울역 출발 기준 소요시간 | 정적 데이터 |
| 태그 | 사용자 투표 기반 도시 특성 | 사용자 투표 |
| 좋아요/싫어요 | 도시 추천 비율 | 사용자 투표 |

**색상 코딩 체계**:

| 색상 | 의미 | 적용 |
|------|------|------|
| GREEN | 좋음 / 긍정 | 안전 높음, PM2.5 좋음, 인터넷 빠름 |
| YELLOW | 보통 / 주의 | PM2.5 보통, 인터넷 보통 |
| RED | 나쁨 / 부정 | PM2.5 나쁨, 안전 낮음 |

**PM2.5 기준**: 0~15 좋음 / 16~35 보통 / 36~75 나쁨 / 76+ 매우나쁨
**인터넷 기준**: 100+Mbps 빠름 / 50~99 보통 / 30~49 느림 / <30 매우느림
